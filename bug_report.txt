Основные пункты баг репорта:
1. Описание (должно содержать информацию, отвечающую на вопросы: Что? Где? При каких обстоятельствах?)
2. Шаги воспроизведения
3. Фактический результат
4. Ожидаемый результат

Так же баг-репорт может содержать:
1. ID
2. Серьезность и приоритет
3. Автор
4. Прикрепленные файлы
